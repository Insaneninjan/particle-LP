# Modern LP

## **Three.js**
[OFFICIAL](https://threejs.org/)
3Dのコンテンツを表示するライブラリ

# Steps
1. rendererの作成
2. sceneの作成
3. cameraの作成
4. Meshの作成
    a. geometoryの設定
    b. materialの設定
4. rendering

# Terms
- Geometory
形状と形状の幅・奥行き・高さを設定

- Material
質感とハイライト・カラー色を設定

- Mesh
Geometory X Materialでできたポリゴンの集まり
